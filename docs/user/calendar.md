# Calendar & Events Module

## Overview
The Calendar module manages scheduling, events, and time-based activities for teams and individuals.

## Features
- **Calendar View** - Monthly, weekly, daily views
- **Event Management** - Create and manage events
- **Meeting Scheduling** - Team coordination
- **Event Categories** - Color-coded organization
- **Reminders** - Automated notifications

## Key Workflows

### Creating Events
1. Go to Calendar
2. Click on desired date/time
3. Fill in event details:
   - Title and description
   - Start/end times
   - Location (physical or virtual)
   - Attendees
   - Category/color
4. Set reminders
5. Save the event

### Managing Recurring Events
1. Create event as usual
2. Set recurrence pattern:
   - Daily, weekly, monthly
   - Custom intervals
   - End date or occurrences
3. Save to create series

### Inviting Attendees
1. Add participants to event
2. Set their roles:
   - **Required**: Must attend
   - **Optional**: Can attend
   - **Resource**: Room/equipment
3. Send invitations
4. Track RSVP responses

### Using Calendar Views
- **Month View**: Overview of all events
- **Week View**: Detailed weekly schedule
- **Day View**: Hour-by-hour breakdown
- **Agenda View**: List of upcoming events
