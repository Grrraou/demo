# Sales Management Module

## Overview
The Sales module manages the complete sales pipeline from quotes to deliveries and invoicing.

## Features
- **Quotes** - Price estimates for customers
- **Orders** - Confirmed sales orders
- **Invoices** - Billing documents
- **Deliveries** - Shipping and fulfillment
- **Sales Dashboard** - Performance metrics

## Navigation
Access via: Main Menu → Sales

## Key Workflows

### Creating Quotes
1. Go to Sales → Quotes
2. Click "Create Quote"
3. Select customer
4. Add products/services
5. Set pricing and terms
6. Send to customer

### Converting Quotes to Orders
1. Open the quote
2. Click "Convert to Order"
3. Confirm details
4. Set delivery date
5. Save the order

### Managing Orders
1. View order status
2. Update quantities
3. Schedule deliveries
4. Track fulfillment

### Creating Invoices
1. Select completed orders
2. Click "Create Invoice"
3. Verify amounts
4. Set payment terms
5. Send to customer

### Processing Deliveries
1. Go to Sales → Deliveries
2. Create delivery from order
3. Set shipping details
4. Track delivery status
5. Confirm receipt

## Status Tracking
- **Quote Status**: Draft, Sent, Accepted, Rejected, Expired
- **Order Status**: Pending, Confirmed, Processing, Shipped, Delivered
- **Invoice Status**: Draft, Sent, Paid, Overdue
- **Delivery Status**: Pending, Shipped, In Transit, Delivered

## Permissions Required
- View Sales: See sales data
- Edit Sales: Create/modify sales documents
- Admin Sales: Full access

## Tips
- Follow the pipeline: Quote → Order → Invoice → Delivery
- Keep customer information updated
- Use templates for common quotes
- Track sales performance regularly
